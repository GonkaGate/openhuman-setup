# Documentation

This directory tracks the implementation contract and shipped behavior for the
OpenHuman setup utility.

- `specs/openhuman-setup-prd/spec.md` is the product requirements document.
- `specs/openhuman-setup-prd/implementation-plan.md` is the current execution
  plan.
- `specs/openhuman-setup-prd/release-proof.md` separates automated proof from
  the manual live smoke gate.
- `specs/openhuman-setup-prd/tasks.md` is the Codex Goal execution checklist.
- `how-it-works.md` describes the intended setup flow.
- `security.md` records secret-handling constraints.
